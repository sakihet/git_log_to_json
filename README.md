# git_log_to_json

print git log in json

[![Gem Version](https://badge.fury.io/rb/git_log_to_json.svg)](https://badge.fury.io/rb/git_log_to_json)

## Installation

```
gem install git_log_to_json
```


## Usage

```
git_log_to_json <path-to-git-repo>
```
